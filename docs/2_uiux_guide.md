# 🎨 [POI 随手记] UI/UX 交互与设计规范文档 (最终版)

## 1. 设计核心原则

* **地图优先 (Map-Centric)**：地图是所有信息的载体，UI 元素应作为悬浮层，减少对地图视野的遮挡。
* **单手掌控 (Thumb-Friendly)**：核心交互（新增、保存、定位）必须位于大拇指操作热区。
* **即时反馈 (Instant Response)**：针对录入行为提供毫秒级反馈，异步处理高耗时任务（如上传）。
* **离线感知 (Offline Awareness)**：明确提示数据存储状态，消除用户对断网丢数据的焦虑。

## 2. 视觉系统 (Visual System)

### 2.1 配色方案 (Color Palette)

| 类别 | 色值 (Hex) | 场景示例 |
| --- | --- | --- |
| **品牌主色** | `#3B82F6` | 按钮背景、当前定位点、选中状态、进度条。 |
| **状态绿色** | `#10B981` | 保存成功提示、已同步状态。 |
| **警告红色** | `#EF4444` | 删除动作、录入项缺失、未授予定位权限。 |
| **中性深色** | `#111827` | 主要标题、正文。 |
| **中性浅色** | `#6B7280` | 次要文字、经纬度数字、空状态描述。 |
| **背景色** | `#F3F4F6` | 非地图页面的背景，如设置或列表。 |

### 2.2 字体与层级 (Typography)

* **字体族**：优先调用系统原生字体（iOS: SF Pro; Android: Roboto）。
* **标题 (H1)**：18px / Semi-bold (POI 名称)。
* **正文 (Body)**：15px / Regular (地址、备注)。
* **标注 (Caption)**：12px / Regular (创建日期、坐标、存储状态)。

## 3. 核心交互组件 (Component Design)

### 3.1 地图标记 (Marker)

* **视觉规范**：采用水滴形状，中间嵌入分类图标。
* **聚合态 (Clustering)**：
* 当点位重叠时，显示带有数字的圆形聚合点。
* 颜色随数量梯次变化（1-10 浅蓝，10-100 深蓝，100+ 紫色）。


* **选中态**：Marker 尺寸放大 1.2 倍，下方出现半透明投影（Elevation）。

### 3.2 底部抽屉 (Bottom Sheet)

* **三段式状态**：
1. **极简态 (Collapsed)**：点击 Marker 弹出，高度 20%，显示名称和导航按钮。
2. **展开态 (Expanded)**：向上滑动展示完整图片、分类、备注、经纬度。
3. **全屏态 (Full)**：查看长备注或多图轮播时。


* **自动避让**：抽屉弹出时，地图中心自动向上平移，确保 Marker 露在抽屉上方。

### 3.3 浮动操作按钮 (FAB)

* **新增按钮**：位于右下角，品牌主色，圆形，直径 56px。
* **辅助按钮**：定位、图层切换。位于右侧垂直排列，白色背景，深色图标。

## 4. 用户交互流程优化 (User Flow)

### 4.1 录入流程 (3 秒极速录入)

1. **触发表单**：点击“+”按钮。
2. **即时捕获**：系统自动获取当前 GPS，用户点击拍照。
3. **快速填写**：仅强制要求输入“名称”，分类默认填充。
4. **异步保存**：点击“保存”，表单立即关闭回到地图。Rust 在后台压缩并上传图片，并在顶部状态栏显示微小的进度动画。

### 4.2 列表与地图切换

* 提供平滑的 **Shared Element Transition (共享元素过渡)**。点击列表中的图片，图片平滑移动并放大至详情页顶部。

---

## 5. 极端环境与异常设计 (Edge Cases)

### 5.1 弱网/离线设计

* **视觉提示**：网络断开时，定位点颜色由蓝色变为灰色。
* **同步队列**：待上传的 POI 在列表视图中显示“等待同步”图标。

### 5.2 强光模式 (High Contrast)

* 地图底图应具备反差明显的样式。
* 关键信息（如 Marker 上的标签）必须带有 1px 的反色描边，确保在任何地形背景下清晰可辨。

---

## 6. 交互热区规范 (Hit Targets)

* **最小热区**：44px x 44px。
* **安全距离**：所有交互元素距离屏幕边缘至少 16px，底部预留刘海屏安全区。

---

## 7. 页面信息架构图 (IA)

* **一级页面**：地图主视图。
* **二级页面**：地点详情页、搜索结果列表、新增表单、**登录/注册页**。
* **三级页面**：**设置页面**（含图床配置、账号管理）、离线地图下载 (区域选择交互)、图片全屏预览。

---

## 8. 新增交互视觉补充

### 8.1 离线区域选择器 (Area Picker)
* **蒙层效果**：进入下载模式后，地图变暗，屏幕中央出现一个 16:9 的镂空高亮框。
* **缩放控制**：锁定缩放级别在 5-13 级之间。
* **动态估算**：屏幕底部实时显示该区域下载 0-12 级瓦片的预计文件大小（如：“预计 45.2 MB”）。
* **确认下载**：点击悬浮在选框下的“开始下载”气泡。

### 8.2 安全输入 (Key Input)
* **掩码切换**：API Key 输入框右侧带“眼睛”图标，点击切换显示/遮蔽密钥，防止录屏泄露。
